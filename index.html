Installation Instructions

Windows Installation

  • After downloading, place the application in an easily accessible folder or on your Desktop
  • Double click or right-click and select 'Open' to run the application
  • You only need to download once - the application can be used until the last day of the exam

macOS Installation

  • After downloading, extract the file
  • Run 'abbs_exam.app' by double-clicking or right-click and select 'Open'
  • If you see a warning, follow these steps:
  • Go to System Settings → Security tab → General
  • Look for the message "abbs_exam.app was blocked from opening" at the bottom
  • Click 'Open Anyway' button
  • Launch the application again
Note: The security warning is normal for macOS applications downloaded from the internet. The app is safe to use.
